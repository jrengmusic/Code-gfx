# GFX - Dark Cyan Theme for VS Code

A dark theme with cyan/teal accents, inspired by REAPER JSFX IDE default color scheme and [OBLIVION-GFX](https://gmunk.com/OBLIVION-GFX).

## Features

- **Dark backgrounds** with subtle blue/cyan tints
- **Vibrant cyan** highlights for active elements
- **High contrast** syntax highlighting optimized for readability
- **Consistent terminal** background matching WezTerm aesthetics
- **Clean UI** with minimal borders and smooth gradients

## Color Palette

- **Editor Background:** Pure black (`#000000`)
- **Sidebar Background:** Dark blue-gray (`#081018`)
- **Terminal Background:** Dark cyan (`#050A0D`)
- **Accent Colors:** Cyan (`#00c8d8`), Teal (`#67dfef`), Orange (`#fc704c`)

## Installation

1. Open **Extensions** in VS Code/VSCodium
2. Search for **"GFX"**
3. Click **Install**
4. Press `Cmd+K Cmd+T` (Mac) or `Ctrl+K Ctrl+T` (Windows/Linux)
5. Select **GFX** from the theme list
